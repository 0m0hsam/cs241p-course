Advice in Reading

Some of the packages and modules reading gets a little technical. We will work through these ideas together. Through all the details, don't lose sight of the most important topics. They are:

  • Classes can contain variables whose data type is another class (composition)

  • Classes may have data that we want to treat as internal or private, so we want to limit its exposure to and access from other classes.

  • When making classes, we are starting to think about making reusable components. Thus, to make our code easier to use, we can put it in different files and folders.
